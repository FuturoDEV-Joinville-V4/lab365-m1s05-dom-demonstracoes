Atividade DOM: Criar e Remover Elementos

Objetivo: Criar e remover elementos HTML dinamicamente na página usando JavaScript.

Sua Tarefa (no atividade_estrutura.js):

  1. Selecione os botões "Adicionar Parágrafo" (id="btnAddParagraph") e "Remover Último Parágrafo" (id="btnRemoveParagraph").
  2. Selecione a área dinâmica (id="area-dinamica") onde os parágrafos serão adicionados/removidos.
  3. **Para o botão "Adicionar Parágrafo":**
    • Adicione um ouvinte de evento de clique.
    • Dentro da função, crie um novo elemento <p>.
    • Defina o textContent do novo parágrafo (ex: "Parágrafo adicionado dinamicamente!").
    • Adicione o novo parágrafo como filho da div#area-dinamica.
  4. **Para o botão "Remover Último Parágrafo":**
    • Adicione um ouvinte de evento de clique.
    • Dentro da função, verifique se a div#area-dinamica possui algum parágrafo (ou filho).
    • Se sim, remova o **último filho** da div#area-dinamica.

Área de Manipulação Dinâmica

Nenhum parágrafo ainda. Adicione um!

Escreva o código JavaScript no arquivo atividade_estrutura.js. Acompanhe os resultados e possíveis erros no console (F12).
