Demonstração DOM: Modificar Elementos

Recuperar Elementos Listar Elementos Modificar Elementos
Objetivo: Demonstrar como alterar dinamicamente o conteúdo de texto, HTML ou o valor de elementos da página usando JavaScript.

Nesta demonstração, vamos:

  1. Modificar o textContent do parágrafo com id="texto-para-mudar" para "O texto deste parágrafo foi alterado pelo JS!".
  2. Alterar o innerHTML da div com id="area-html" para incluir um novo HTML (ex: um <strong> e um link).
  3. Definir um novo value para o campo de input com id="input-exemplo" para "Valor Modificado".
  4. (Opcional) Usar um botão para acionar todas essas modificações.
