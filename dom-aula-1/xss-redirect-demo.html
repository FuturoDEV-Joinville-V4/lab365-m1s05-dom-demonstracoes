Simulação de Ataque XSS: Redirecionamento

Este exemplo demonstra como um script malicioso injetado pode forçar o navegador a redirecionar o usuário para uma página diferente.

1. Injetando Script de Redirecionamento (Vulnerável)

Cole o código de ataque no campo abaixo e clique em "Ativar Ataque".

Código de Ataque para colar:

<script>window.location.href = 'pagina-maliciosa-fake.html';</script>

**Atenção:** Este é um exemplo didático. Em um ataque real, a página de destino seria controlada pelo atacante.
