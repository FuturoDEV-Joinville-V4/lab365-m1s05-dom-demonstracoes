Demonstração DOM: Recuperar Elementos

Recuperar Elementos Listar Elementos Modificar Elementos
Objetivo: Demonstrar como selecionar e acessar o conteúdo de diferentes elementos HTML no DOM utilizando os métodos JavaScript.

Nesta demonstração, vamos:

  1. Selecionar o <h1> principal usando querySelector() e logar seu conteúdo.
  2. Acessar o parágrafo com id="paragrafo-unico" usando getElementById() e logar seu textContent.
  3. Obter todos os itens da lista (<li>) com a classe item-lista usando querySelectorAll() e explorar a coleção.
  4. Capturar todos os elementos <span> usando getElementsByTagName().
  5. Verificar o conteúdo (inclusive HTML) de uma div com a classe info-box usando getElementsByClassName() e sua propriedade innerHTML.
  6. Acessar o value de um campo de input com id="nome-usuario".
